## Capture tips

- Use a clean window at ~1440×900 or retina 2× for crisp GitHub rendering.
- Prefer light theme shots that match the website unless the feature is dark-only.
- Crop chrome noise (desktop wallpaper, unrelated apps).
- Keep sensitive tokens, emails, and private chat content out of frames.
- Keep README videos under ~50MB when possible so clones stay reasonable.
